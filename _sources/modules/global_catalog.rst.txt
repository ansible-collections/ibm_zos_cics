global_catalog -- Create and initialize CICS® global catalog.
=============================================================



.. contents::
   :local:
   :depth: 1


Synopsis
--------
- Create, update, and remove a CICS® global catalog dataset to be used by a CICS® region, achieved by running the IDCAMS and DFHRMUTL programs.





Parameters
----------


     
location
  The name of the dataset to be used as the global catalog dataset.


  | **required**: True
  | **type**: str


     
sdfhload
  The location of SDFHLOAD for the CICS install to be used to create the global catalog.


  | **required**: True
  | **type**: str


     
space_primary
  Specifies the size of the global catalog dataset. Note, this is just the value; the unit is specified with space_type.


  | **required**: False
  | **type**: int
  | **default**: 5


     
space_type
  Specifies the unit to be used for the global catalog size. Note, this is just the unit; the value is specified with space_primary.


  | **required**: False
  | **type**: str
  | **default**: M
  | **choices**: M, K, REC, CYL, TRK


     
state
  The state the catalog will end up in after the task has run.


  | **required**: True
  | **type**: str
  | **choices**: initial, absent, cold, warm




Examples
--------

.. code-block:: yaml+jinja

   
   - name: Initialize a global catalog
     ibm.ibm_zos_cics.global_catalog:
       location: "CICS.INSTALL.REG01.DFHGCD"
       sdfhload: "CTS560.CICS730.SDFHLOAD"
       state: "initial"

   - name: Initialize a large catalog
     ibm.ibm_zos_cics.global_catalog:
       location: "CICS.INSTALL.REG01.DFHGCD"
       sdfhload: "CTS560.CICS730.SDFHLOAD"
       space_primary: 100
       space_type: "M"
       state: "initial"

   - name: Set AUTO_START_OVERRIDE record to be AUTOASIS
     ibm.ibm_zos_cics.global_catalog:
       location: "CICS.INSTALL.REG01.DFHGCD"
       sdfhload: "CTS560.CICS730.SDFHLOAD"
       state: "warm"

   - name: Set AUTO_START_OVERRIDE record to be AUTOCOLD
     ibm.ibm_zos_cics.global_catalog:
       location: "CICS.INSTALL.REG01.DFHGCD"
       sdfhload: "CTS560.CICS730.SDFHLOAD"
       state: "cold"

   - name: Delete global catalog
     ibm.ibm_zos_cics.global_catalog:
       location: "CICS.INSTALL.REG01.DFHGCD"
       sdfhload: "CTS560.CICS730.SDFHLOAD"
       state: "absent"
